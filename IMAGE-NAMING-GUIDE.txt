  IMAGE NAMING GUIDE — heybharat portfolio
═══════════════════════════════════════════════════════════════════

  Rename your image files to the exact names below, then drop them
  into the same folder as index.html. Files are already wired up in
  the code — no editing needed once filenames match.

  All image filenames are LOWERCASE, hyphen-separated, .jpg.

───────────────────────────────────────────────────────────────────
  TIPS
───────────────────────────────────────────────────────────────────

  - 16:9 images = 1600×900 or 1920×1080 — no cropping at the edges.
  - 1:1 (square) images = 1080×1080 — for Instagram posts.
  - 9:16 (portrait) images = 1080×1920 — for stories and reels.
  - 16:6 wide images = 1920×720 — for hero banners.
  - All file names are lowercase, no spaces, hyphens between words.
